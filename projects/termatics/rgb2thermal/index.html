RGB2Thermal - Termatics

This project aimed to develop a machine learning pipeline for predicting thermal images from RGB inputs. By leveraging multiple datasets and experimenting with different modeling approaches, significant progress was made despite challenges in data generation and alignment.

Data Challenges and Solutions

Starting with a small dataset of 555Ã—4 RGB-thermal pairs, I expanded it through simulation using AirSIM and Unreal Engine. Real-world datasets (e.g., FLIR) were also incorporated for better diversity and accuracy.

Modeling Approaches

Experimented with unpaired CycleGAN and paired Pix2Pix models. Pix2Pix achieved better results, but distortions highlighted the need for enhanced data alignment.

Data Alignment

Developed a pipeline to align RGB and thermal images at a pixel level. This improved the quality of predictions significantly, demonstrating the importance of pre-processing in machine learning workflows.
