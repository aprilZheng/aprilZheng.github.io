2D to 3D Thermal Modeling - Termatics

This project explores the reconstruction of a 3D thermal model using pairs of 2D RGB and thermal images. The goal was to integrate machine learning and 3D modeling techniques for accurate thermal visualization. Despite challenges, the project offered valuable insights into thermal image processing and 3D reconstruction workflows.

Color Calibration

Thermal cameras often map temperature ranges dynamically, causing color inconsistencies between images. To address this, I implemented a calibration process that standardized color mapping across images, making them suitable for 3D reconstruction.

Experimenting with Gaussian Splatting

I tested Gaussian Splatting, a method for reconstructing 3D models, using RGB images to evaluate its feasibility for city-scale 3D reconstruction. The results, while computationally expensive, provided key insights into the limitations of this approach for thermal data integration.

Generating a Base 3D Model

Due to the limitations of existing tools like Google 3D Tiles and Cesium for editable models, I used OpenStreetMap's 3D map data to create a basic framework for thermal mapping

UV Mapping Calibrated Thermal Images

The calibrated thermal images were manually UV mapped onto the 3D model to visualize thermal data. While automation was considered, manual adjustments were made to meet tight competition deadlines. This step showcased the potential of combining thermal imagery with 3D modeling techniques.

While the results of this project revealed challenges in thermal 3D modeling, it also provided a foundation for future improvements. With further exploration, automating workflows and refining reconstruction techniques could lead to more accurate and scalable solutions.
