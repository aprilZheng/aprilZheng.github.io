Graffiti Simulation in Virtual Reality

This VR graffiti simulation project allows users to collaborate and create virtual graffiti art on a wall. It integrates several key functionalities, including spray painting, sketch projection, collaborative work with networking features, and realistic interactions such as using a ladder or mixing paint. The project creates an engaging environment for users to explore their creativity while working together in a virtual space.

Collaborative Networking

One of the core features of this project is the ability for users to collaborate in a shared virtual space. Players can join the same room, interact with the environment, and paint together in real-time. Networking is handled by the Ubiq Messaging package, ensuring seamless synchronization between users' actions and the objects in the environment.

Spray Can Functionality

The spray can tool lets users pick colors from a dedicated panel and apply strokes on 3D models. The paint strokes are synchronized across all users, ensuring everyone sees the same artwork in real-time.

Sketch Projection

Users can select an image or draft to project onto the wall. This visual cue helps guide their painting process. In this case, the sketch is an outline that users can follow while spraying their artwork.

Paint Mixing and Roller

To add a more collaborative element, users can mix different colors of paint into a large bucket. This process brings users together, as they work together to create the ideal shade. Once the paint is mixed, users can use a roller to apply the paint to the wall. The roller tool is synchronized for all users, allowing multiple people to paint on different areas of the wall simultaneously.
