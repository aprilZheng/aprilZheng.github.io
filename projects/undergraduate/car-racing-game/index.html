2D Racing Car Game

This project is a Unity-based 2D racing game designed during my undergraduate studies. The game offers a simple yet engaging experience, with a focus on procedural map generation and dynamic gameplay elements.

Features

  • Procedural Map Generation: Maps are generated dynamically, ensuring each playthrough is unique.
  • Interactive Gameplay: Players navigate a track filled with dynamic obstacles, with penalties applied for collisions.
  • Performance Tracking: A ranking system based on driving time and distance adds a competitive edge.

Reflection

This project allowed me to strengthen my skills in Unity and C#, particularly in procedural content generation and game mechanics design. It also introduced me to memory optimization techniques and gameplay balancing, laying the foundation for more advanced projects.
